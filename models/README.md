# Speech Recognition Models

This project uses the VOSK offline speech recognition engine for Hindi language support.

Due to large file size, the speech model is not included in this GitHub repository.

##  Download Instructions

1. Visit:
https://alphacephei.com/vosk/models

2. Download:
vosk-model-small-hi-0.22 (or latest Hindi model)

3. Extract the folder into this directory:

models/vosk-model-small-hi-0.22/

Final structure should look like:

models/
 └── vosk-model-small-hi-0.22/
     ├── conf/
     ├── graph/
     └── model files
